Detailed Description

template<typename Number, typename Converter>
class hypro::reachability::hypro::BoxT< Number, Converter >

The class which represents a box.

A box is represented by two points, its maximal and its minimal vertex.

Template Parameters
NumberThe used number type.
ConverterThe used converter. @{
