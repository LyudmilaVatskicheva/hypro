Detailed Description

template<typename Number, typename Converter>
class hypro::ConstraintSetT< Number, Converter >

A class representing a plain constraint set.

A constraint set in this case is a datastructure similar to a H-Polytope but without any functionality. The idea is just to have a container for data received from the parser or as an intermediate type.
